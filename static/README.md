# IDLIX Stream Player

Simple HTML5 video player for testing HLS streams.

## Features

✅ **Video.js HLS Player** - Industry standard HTML5 player
✅ **API Integration** - Fetch video info from IDLIX API
✅ **Quality Selector** - Choose from available variants
✅ **Direct M3U8** - Play any M3U8 URL directly
✅ **Responsive Design** - Works on all devices
✅ **Beautiful UI** - Modern gradient design

## Access

```
http://localhost:8080/player
```

## Usage

### Method 1: Via API (Recommended)

1. Enter IDLIX movie URL
2. Click "Fetch Video Info"
3. Video will auto-play with quality options

### Method 2: Direct M3U8

1. Paste M3U8 URL in second input
2. Click "Play Direct M3U8"
3. Video plays immediately

## Features

- **Adaptive Streaming** - Auto quality based on bandwidth
- **Quality Selection** - Manual quality control
- **Error Handling** - User-friendly error messages
- **Loading States** - Visual feedback
- **Video Info Display** - Show metadata

## Technology

- Video.js 8.10.0
- HLS.js integration
- Responsive CSS
- Fetch API for backend communication
